Laravel Test con GitHub Copilot

Datos identificativos

Título:

Laravel Test con GitHub Copilot

Etapa:

Ciclo Formativo de Grado Superior de Desarrollo de Aplicaciones Web

Curso:

2.º

Módulo:

Desarrollo web en entorno servidor

Temporalización y relación con la programación

Esta situación de aprendizaje permitirá a los estudiantes aplicar GitHub Copilot de manera efectiva para generar pruebas funcionales que evalúen el correcto funcionamiento de un controlador de recursos en proyectos Laravel, lo que mejorará la calidad del código y el proceso de desarrollo web.

La presente situación de aprendizaje requiere de un mínimo de seis horas, en sesiones de dos horas. Sería deseable iniciarla cuando el alumnado haya abordado los contenidos correspondientes a los controladores de recursos.

Parte de las tareas podrán ser realizadas por el alumnado en su propio domicilio, preferentemente a través de la máquina virtual facilitada al inicio del curso.

En cuanto a la finalidad de los aprendizajes, esta situación de aprendizaje dota al profesorado de recursos educativos variados que contribuyen a que su alumnado:

  • Completar planes de pruebas verificando el funcionamiento de los componentes software desarrollados, según las especificaciones.
