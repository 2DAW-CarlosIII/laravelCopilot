Método update

Para configurar un test para el método update, vamos a comenzar con el siguiente comentario:

    /** Genera un test funcional para el método update del controlador de recursos CustomerController
     * teniendo en cuenta que el método update acepta una recurso del modelo Customer
     */

Inicialmente, el comentario anterior únicamente nos sugiere la cabecera del método test_update

public function test_update(): void

No obstante, si lo aceptamos y pulsamos sobre el intro al final de la línea, veremos que nos sugiere el resto del método.

    public function test_update(): void
    {
        $response = $this->put('/api/customers/1', [
            'first_name' => 'John',
            'last_name' => 'Doe',
            'job_title' => 'Developer',
            'city' => 'Cartagena',
            'country' => 'Spain',
            'user_id' => 1
        ]);
        $response->assertStatus(200)->assertJsonStructure(['data' => [
            'id',
            'first_name',
            'last_name',
            'job_title',
            'city',
            'country',
            'user_id',
            'created_at',
            'updated_at'
        ]]);
    }
